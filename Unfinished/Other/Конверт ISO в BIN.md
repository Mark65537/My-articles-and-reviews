# Как конвертировать ISO в BIN CUE

## bchunk

Вроде как можно через эту утилиту используя wsl но у меня выдавало ошибку

```bash
bchunk input.iso output.bin output.cue
```

## В ручную

1. записываем образ на диск
2. снимаем образ с помощью ImgBurn
