## Структура игры

Движок игры мы будем для удобства называть Dirty engine. У него есть 3 версии, для удобства мы их так и будем называть 1, 2, 3. Версии будут идти от простой для декомпиляции до самой сложно.

Причем доходило даже до абсурдного, например, для игры Casanova love mission с разрешением экрана 240x320, использовался движок версии 1, а для разрешения 176x220, версии 2.

## Сценарий

Если у файла нет кода языка, то это файл по умолчанию, обычно там английский язык или русский если игру перевели. Есть несколько версий этого файла, в зависимости от версии движка, но об этом поговорим чуть позже. 

В некоторых играх, например: Casanova секс студентки, в этом файле прописаны только заголовки(см ___), в остальных играх прописаны все диалоги и заголовки
