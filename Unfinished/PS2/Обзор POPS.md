# Обзор POPS

## Патчи

PATCH_1.BIN - если в игре есть проблемы c музыкой/голосом!
PATCH_2.BIN - если в игре есть проблема со звуком (создан для серии Colony)
PATCH_3.BIN - нужно использовать, если не помог PATCH_1.BIN
PATCH_4.BIN - даёт большую совместимость для игр, решает проблему со скоростью и.т.д.
PATCH_5.BIN - создан для Resident Evil, но подойдёт и для других игр.
PATCH_6.BIN - отключает BIOS логотип, иногда помогает при запуске игр, когда вместо биос грузиться чёрный экран.
PATCH_7.BIN - исправляет графику в некоторых играх, в первую очередь там где пропадает фон.
PATCH_8.BIN и PATCH_9.BIN - решают проблему с экраном!

## Читы

Полезные коды, которые можно добавить в файл CHEATS.txt

$COMPATIBILITY_0x?? - вместо ?? используем число от 01 до 09. Данный чит используется вместо патчей.
$COMPATIBILITY_0x05
$SMOOTH - сглаживание
$HDTVFIX - для игры через компонент
$NOPAL - если проблема с PAL версиями
$480p - поддержка прогрессивной развёртки в NTSC играх, правда разницы не заметил.
$CACHE1 - если игра тормозит.
$CODECACHE_ADDON_0 - большая совместимость с играми.
$SUBCDSTATUS - альтернатива ($COMPATIBILITY_0x05), в общем исправляет звук в некоторых играх.
$MUTE_CDDA - отключает поддержку CDDA (CD аудио)
$MUTE_VAB - отключает звук (rawc)
$WIDESCREEN - поддержка широкого экрана.
$ULTRA_WIDESCREEN - поддержка широкого экрана.
$EYEFINITY - поддержка широкого экрана.
$SET_TIMINGS - правит графику в некоторых играх
$LOAD_TIMINGS - правит графику в некоторых играх
Rumble Always On (Pad 1) - вибрация
$00507028 00000001
Rumble Always On (Pad 2) - задействовать вибрацию
$005070B8 00000001
$D2LS - аналоговый режим
$D2LS_ALT - аналоговый режим
$NOVMC0 - отключает карту памяти 1
$NOVMC1 - отключает карту памяти 2, код полезен для игры TENKA

Новый чит-код $SUBCDSTATUS - альтернатива ($COMPATIBILITY_0x05), в общем исправляет звук в некоторых играх.
